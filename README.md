### 2. Calculadora de Combustível (UFMG)
- **Caminho:** `/calculadora-combustivel`
- **Descrição:** Script que calcula a viabilidade entre Gasolina e Etanol com base na regra de eficiência de 70%.
- **Destaque:** Meu primeiro projeto desenvolvido na UFMG em 2023, focado em lógica matemática e modularização com funções em C.
